# 📚 Projeto Livros

Aplicação fullstack para gerenciar livros favoritos.  
Desenvolvida como parte de um teste prático, com backend em Node.js + Express e frontend em React (Vite).

---

## 🚀 Tecnologias usadas

### **Backend**
- Node.js
- Express
- CORS
- Railway (deploy)

### **Frontend**
- React (Vite)
- Axios
- CSS simples para estilização
- Vercel (deploy)
